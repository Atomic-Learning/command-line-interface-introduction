A command line interface (CLI) is a text-based interface for interacting with a computer program or operating system. To use a CLI, you typically type commands into a terminal or console window and press Enter to execute them. The interface is able to display output from the executed commands. All manner of tasks can be performed using a CLI including file management, system configuration, and running applications.

Use Cases

A CLI is, at first glance, less user-friendly than a graphical user interface (GUI) as you need to know commands and their syntax, rather than having menus and buttons presented for you to click on. However, a CLI offers several advantages:

  • Efficiency: For experienced users, a CLI can be much faster than a GUI. Commands can be executed quickly without the need for navigating through menus or clicking buttons.
  • Reproducibility: CLI commands can be saved in files known as scripts and executed by a user, which is useful for repetitive tasks.
  • Automation: Scripts can be scheduled to run automatically at specified times or intervals, or in response to specific events.
  • Remote Access: When accessing another computer remotely, there will not always be a GUI available. In many cases, a CLI is the only option.
  • Resource Usage: CLIs typically consume fewer system resources than GUIs, which can be beneficial when working on low-powered devices or over slow network connections.

Examples of CLIs

There are several different CLIs which are commonly used. Some of these are part of operating systems:

  • Linux: Many Linux systems use Bash by default.
  • macOS: The default CLI in macOS is zsh.
  • Windows: In Windows, Command Prompt and PowerShell are widely used.

Some programs also provide their own CLI interfaces, such as the Git CLI for version control. Some commands will be common across different CLIs, while others may be specific to a particular implementation.

CLI vs Terminal vs Shell

These terms are related but not identical:

  • CLI (Command Line Interface): A text-based way to interact with software by typing commands. A CLI is the interaction style or interface.
  • Terminal: The application window you use to type and view command output (for example, Windows Terminal, Terminal on macOS, or a Linux terminal app).
  • Shell: The command interpreter running inside the terminal (for example, Bash, zsh, PowerShell). The shell reads your commands and executes them.

In short: you open a terminal, which runs a shell, and you use that shell through a CLI.
